# Roadmap

Baseline: `0.7.0`

## 0.7.0

- Native Qt/QML desktop shell.
- Rust core/ipc/service workspace.
- Compact subscription-first UI.
- Clipboard import skeleton.
- Mock ping, connection, speed, and traffic.
- Local package script.

## 0.2.0

- Real subscription fetching.
- VLESS, Trojan, Shadowsocks, Hysteria2, TUIC, WireGuard, AmneziaWG parsing in Rust.
- Local encrypted subscription storage.
- UI bridge from Rust core to Qt model.

## 0.3.0

- sing-box adapter.
- Generated config preview in advanced settings.
- Process lifecycle and logs.
- System proxy mode.

## 0.4.0

- TUN mode.
- Real connection status.
- Speed counters from engine/runtime.
- Auto reconnect.

## 0.5.0

- Rust Windows service with named pipe IPC.
- Service install/start/stop/status.
- Desktop fallback disabled for privileged actions.

## 0.6.0

- WireGuard and AmneziaWG adapters.
- Runtime config cleanup.
- Engine diagnostics.

## 0.7.0

- Whole computer, selected apps only, and except selected apps routing.
- App picker.
- WFP/firewall groundwork.

## 0.8.0

- Kill switch.
- DNS leak protection.
- IPv6 policy.
- Watchdog and rollback.

## 0.9.0

- Installer.
- Auto update.
- Code signing preparation.
- Support export without secrets.

## 1.0.0

- Stable release.
- Security review.
- Abuse-resistance server hooks.
- Signed production package.
